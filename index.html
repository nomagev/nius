    TERMINAL HACKER NEWS BROWSER V.1.0
# System Status: Online

Nius is a minimalist, distraction-free Hacker News client built for the terminal. It provides a retro-styled interface for staying updated with tech news without the noise of a modern browser.

> FEATURES

  • Typewriter Visual Effects
  • Markdown-to-Terminal Formatting
  • Single-Key Navigation (ESC/B)
  • Ultra-Lightweight API Integration

> NAVIGATION

  • [Number] Open Story & Comments
  • [ESC] Scroll through Comments
  • [B] Return to Feed
  • [Q] Exit Application
